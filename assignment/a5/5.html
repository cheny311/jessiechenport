Project Description

I used code from my classmate Anna Zimmer for the eye-tracking. From there, I layered “unreadable” characters to create an optical illusion: when the system detects the viewer’s eyes on the screen, the message becomes unreadable; when the viewer looks away, the text “can you see me?” is revealed. I used a for-loop to draw strips that add a blur-like effect, and lighter colors to further obscure the text. During the process, I realized that once the foundation was set, it was hard to add more design without weakening the concept. That challenge also taught me the value of “less is more.”
