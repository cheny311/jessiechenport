Project Description

My idea was to make a face that shifts into crying as the mouse moves. I use lerp so the change feels smooth—eyebrows rotate, the mouth stretches, and tears grow longer as t increases. For the background, I added a mousePressed() that randomizes the color. it was fun to play around with the location of the pupil and eyebrow to create different facial expression.
