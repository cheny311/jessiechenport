This is a data portrait that visualizes personal data through an animated character.

The character features:

  • Red hair with individual strands
  • Animated mouth that displays data as a wave form
  • Interactive arms that follow your mouse movement
  • Data values: [90, 67, 90, 55, 82, 100, 80]

The mouth visualization shows the data points with a sinusoidal animation, creating a living, breathing data portrait.
